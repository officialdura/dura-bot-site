Privacy Policy

Last updated: June 10, 2026

1. Information We Collect

Dura ("the Bot") collects and stores the following data solely for functionality purposes:

  • Discord User IDs — for economy balances, leveling, warnings, and ticket identification.
  • Server (Guild) IDs — to store per-server configuration and command registration.
  • Channel IDs — to know where to send welcome messages, logs, and tickets.
  • Role IDs — for auto-role assignment on level-up and member join.
  • Configuration Data — server-specific settings such as prefix alternatives, moderation logs channel, and ticket categories.

We do not collect message content, private messages, or any personal identifiable information beyond what is listed above.

2. How We Use the Data

The data is used exclusively to operate the Bot's features:

  • Awarding XP and tracking levels
  • Managing the virtual economy (balances, shop purchases)
  • Creating and managing support tickets
  • Moderation actions (warnings, timeouts, bans)
  • Auto-moderation (banned word filtering, NSFW image detection)

3. Data Storage & Retention

Data is stored in-memory while the Bot is running. No persistent database or external storage service is used. Data is lost when the Bot restarts. Server administrators may request manual data reset at any time.

4. Data Sharing

We do not sell, share, or distribute any collected data to third parties. Data stays within the Bot's runtime memory and is never transmitted externally except for normal Discord API communication.

5. User Rights

Server administrators may request removal of their server's data by contacting us. Individual users should contact the server administrator where the Bot is used, as we do not maintain a central user database.
